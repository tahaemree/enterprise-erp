# Security Policy

## Supported Versions

We take security seriously. Currently, the following branches are receiving security updates:

| Version | Supported          |
| ------- | ------------------ |
| main    | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please report any security vulnerabilities by opening an issue on GitHub and tagging it with `security`, or by contacting the maintainer directly if the issue is highly sensitive. We aim to respond to security reports within 48 hours.
